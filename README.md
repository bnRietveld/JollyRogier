#### Installation:

1) Make sure you have the latest grunt & karma packages (I assume you already installed nodejs) so run:
``` bash
$ npm install karma -g
$ npm install gulp  -g
$ npm install bower -g
```
2) Intialize all local node & bower packages by running
``` bash
$ npm install
$ bower install
```

#### Builds:

Builds are made using gulp
``` bash
$ gulp
```
